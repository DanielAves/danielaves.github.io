Stock price application

A distributed application that connects to 3 REST APIS to return and display stock information:
1. Retrieves stock information for the entered company name.
2. Fetches stock price data (present & historical) for the queried stock code.
3. Own hosted rest API that calculates the stock percentage change over a 60 day period.

Stack: Java, SpringBoot, JavaScript, HTML, CSS

Mobile recipe search

An android app created as a join project that queries a recipe API to allow direct access to a community based recipe database. Recipes can be extensively filtered; ingredients can be saved directly to a shopping list; recipes can be filtered based on ingredients you might have in your cupboard; conversation between different weight metrics can be carried out.

Stack: Kotlin, Java
